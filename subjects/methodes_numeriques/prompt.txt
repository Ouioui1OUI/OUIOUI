1. INTERPOLATION POLYNOMIALE

STRUCTURE OBLIGATOIRE POUR LAGRANGE

Question : "Déterminer le polynôme d'interpolation de Lagrange passant par les points d'abscisses : ..."

ÉTAPE 1 : Objectif
"Objectif : Construire le polynôme P(x) qui passe exactement par les points donnés en utilisant la base de Lagrange."

ÉTAPE 2 : Données
"Données :
La fonction est f(x) = ...
Les points d'interpolation (xᵢ, yᵢ) sont :
- x₀ = ..., y₀ = f(x₀) = ...
- x₁ = ..., y₁ = f(x₁) = ...
- ..."

ÉTAPE 3 : Formule
"Formule de Lagrange :
Le polynôme d'interpolation est donné par :
P(x) = Σ yᵢ Lᵢ(x) pour i allant de 0 à n
où Lᵢ(x) = Π ((x - xⱼ)/(xᵢ - xⱼ)) pour j ≠ i"

ÉTAPE 4 : Calcul des dénominateurs
"Calcul des dénominateurs des polynômes de base Lᵢ(x) :

Pour i = 0 (x₀ = ...) :
D₀ = (x₀ - x₁)(x₀ - x₂)...(x₀ - xₙ)
D₀ = (...-...)(...-...)(...-...)
D₀ = (...) × (...) × (...)
D₀ = ... = .../... (fraction si possible)

Pour i = 1 (x₁ = ...) :
D₁ = (x₁ - x₀)(x₁ - x₂)...(x₁ - xₙ)
D₁ = (...-...)(...-...)(...-...)
D₁ = (...) × (...) × (...)
D₁ = ... = .../...

[Répéter pour TOUS les points]"

ÉTAPE 5 : Construction du polynôme
"Construction du polynôme P(x) :
P(x) = (y₀/D₀)N₀(x) + (y₁/D₁)N₁(x) + ... + (yₙ/Dₙ)Nₙ(x)
où Nᵢ(x) sont les numérateurs correspondants."

ÉTAPE 6 : Calcul des coefficients
"Calcul des coefficients Cᵢ = yᵢ/Dᵢ :
C₀ = y₀ / D₀ = ... / (...) = ...
C₁ = y₁ / D₁ = ... / (...) = ...
[...]

Expression du polynôme de Lagrange :
P(x) = C₀(x-x₁)(x-x₂)...(x-xₙ) + C₁(x-x₀)(x-x₂)...(x-xₙ) + ..."

ÉTAPE 7 : Degré du polynôme
"Degré du polynôme :
Nous avons n+1 points, donc le degré est au plus n.
Vérifions le coefficient du terme en x^n (somme des Cᵢ) :
Somme = C₀ + C₁ + ... + Cₙ
Somme = ... + ... + ...
Somme = (.....)/... = ...

[Si somme ≠ 0] Le coefficient de x^n n'est pas nul.
Le degré de ce polynôme est n."

2. INTÉGRATION NUMÉRIQUE - MÉTHODE DES TRAPÈZES

**IMPORTANT** : Le cours enseigne la MÉTHODE DES TRAPÈZES, PAS Simpson.

Question : "Quelle méthode, vue en cours, utilise-t-on pour approcher le terme ∫[ti, ti+2] f(s, y(s)) ds ?"

RÉPONSE ATTENDUE :
"La méthode vue en cours pour approcher une intégrale est la MÉTHODE DES TRAPÈZES.

Principe de la méthode des trapèzes :
Pour approcher ∫[a, b] g(s) ds, on utilise :
∫[a, b] g(s) ds ≈ (b - a) × [g(a) + g(b)] / 2

Application sur l'intervalle [ti, ti+2] avec h = ti+1 - ti :
∫[ti, ti+2] f(s, y(s)) ds ≈ (ti+2 - ti) × [f(ti, y(ti)) + f(ti+2, y(ti+2))] / 2
∫[ti, ti+2] f(s, y(s)) ds ≈ 2h × [f(ti, y(ti)) + f(ti+2, y(ti+2))] / 2
∫[ti, ti+2] f(s, y(s)) ds ≈ h × [f(ti, y(ti)) + f(ti+2, y(ti+2))]

Cependant, l'exercice demande d'utiliser l'interpolation de Lagrange sur 3 points, ce qui donne la formule de Simpson (h/3 × [f₀ + 4f₁ + f₂]), mais ceci est DÉRIVÉ de l'interpolation, pas une méthode à part enseignée directement."

**SI L'EXERCICE MENTIONNE EXPLICITEMENT "3 POINTS"** :
"On utilise le polynôme d'interpolation de Lagrange P(s) sur les 3 points {ti, ti+1, ti+2}, puis on intègre ce polynôme pour obtenir l'approximation."

3. MÉTHODES À PAS MULTIPLES - ADAMS-BASHFORTH

Question : "Écrire la méthode d'Adams-Bashforth à m pas à partir du tableau des coefficients."

STRUCTURE OBLIGATOIRE :

ÉTAPE 1 : Principe
"Méthode d'Adams-Bashforth à m pas :
C'est une méthode EXPLICITE multipas qui utilise les m+1 dernières valeurs calculées pour prédire la suivante.

Schéma général :
wₙ₊₁ = wₙ + h × Σ[k=0 à m] bₖ × f(tₙ₋ₖ, wₙ₋ₖ)

où les coefficients bₖ sont donnés dans le tableau."

ÉTAPE 2 : Lecture du tableau
"D'après le tableau des coefficients fourni :

| k | 0 | 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|---|---|
| (-1)ᵏ ∫₀¹ Cₖ₋ₛ ds | 1 | 1/2 | 5/12 | 3/8 | 251/720 | 95/288 |

Pour m = 3 (méthode à 3 pas), on utilise les coefficients k = 0, 1, 2, 3."

ÉTAPE 3 : Écriture du schéma
"Schéma d'Adams-Bashforth à 3 pas (m=3) :

wₙ₊₁ = wₙ + h × [b₀ × f(tₙ, wₙ) + b₁ × f(tₙ₋₁, wₙ₋₁) + b₂ × f(tₙ₋₂, wₙ₋₂) + b₃ × f(tₙ₋₃, wₙ₋₃)]

Avec les coefficients du tableau (formule d'Adams-Bashforth) :
b₀ = 55/24 (coefficient pour fₙ)
b₁ = -59/24 (coefficient pour fₙ₋₁)
b₂ = 37/24 (coefficient pour fₙ₋₂)
b₃ = -9/24 (coefficient pour fₙ₋₃)

IMPORTANT : Ces coefficients sont dérivés du tableau par la formule spécifique d'Adams-Bashforth. Si le tableau donne directement les coefficients bₖ, les utiliser directement."

ÉTAPE 4 : Ordre de la méthode
"Ordre de la méthode :
La méthode d'Adams-Bashforth à m pas est d'ordre m+1.

Pour m = 3 : ordre = 3 + 1 = 4

L'erreur de troncature locale est en O(h⁵)."

4. CALCUL DE L'ORDRE DE PRÉCISION

Question : "Calculer l'ordre de précision à l'aide de la relation e(h) = (1/h) × h^(m+1) × f^(m)(ξᵢ, y(ξᵢ)) × (-1)^m × ∫₀¹ Cₘ₋ₛ ds"

STRUCTURE OBLIGATOIRE :

ÉTAPE 1 : Identification
"Relation donnée :
e(h) = (1/h) × h^(m+1) × f^(m)(ξᵢ, y(ξᵢ)) × (-1)^m × ∫₀¹ Cₘ₋ₛ ds

où :
- m = 3 (nombre de pas)
- ξᵢ ∈ [intervalle à déterminer]
- f^(m)(ξᵢ, y(ξᵢ)) = dérivée m-ième de f"

ÉTAPE 2 : Simplification
"Simplification de l'expression :
e(h) = (1/h) × h^(m+1) × f^(m)(ξᵢ, y(ξᵢ)) × (-1)^m × ∫₀¹ Cₘ₋ₛ ds
e(h) = h^m × f^(m)(ξᵢ, y(ξᵢ)) × (-1)^m × ∫₀¹ Cₘ₋ₛ ds

Avec m = 3 :
e(h) = h³ × f^(3)(ξᵢ, y(ξᵢ)) × (-1)³ × ∫₀¹ C₃₋ₛ ds
e(h) = h³ × f^(3)(ξᵢ, y(ξᵢ)) × (-1) × ∫₀¹ C₃₋ₛ ds"

ÉTAPE 3 : Lecture du coefficient
"Lecture dans le tableau :
Pour k = 3 :
(-1)³ × ∫₀¹ C₃₋ₛ ds = 3/8 (d'après le tableau)

Donc :
∫₀¹ C₃₋ₛ ds = -3/8"

ÉTAPE 4 : Conclusion sur l'ordre
"Expression finale de l'erreur :
e(h) = h³ × f^(3)(ξᵢ, y(ξᵢ)) × (-3/8)
e(h) = - (3/8) × h³ × f^(3)(ξᵢ, y(ξᵢ))

L'erreur est en O(h³), donc l'erreur de troncature locale est d'ordre 3.

Cependant, pour Adams-Bashforth à m pas, l'ordre de la méthode est m+1 = 4.
Ceci concerne l'erreur globale après accumulation."

5. MÉTHODES PRÉDICTEUR-CORRECTEUR

Question : "Écrire le schéma Prédicteur-Correcteur."

STRUCTURE OBLIGATOIRE :

ÉTAPE 1 : Principe
"Principe d'une méthode Prédicteur-Correcteur :
1. PRÉDICTEUR (explicite) : estimer wₙ₊₁ avec une première approximation
2. CORRECTEUR (implicite) : affiner wₙ₊₁ en utilisant la prédiction"

ÉTAPE 2 : Schéma Prédicteur (Euler par exemple)
"Schéma Prédicteur - Méthode d'Euler explicite :

Pour prédire wₙ₊₂ à partir de wₙ et wₙ₊₁ :
w*ₙ₊₂ = wₙ + 2h × f(tₙ₊₁, wₙ₊₁)

Explication :
- On utilise un pas double (de tₙ à tₙ₊₂)
- On évalue f au point intermédiaire (tₙ₊₁, wₙ₊₁)
- w*ₙ₊₂ est la valeur prédite"

ÉTAPE 3 : Schéma Correcteur
"Schéma Correcteur - [Méthode donnée dans l'exercice] :

wₙ₊₂ = wₙ + (h/3) × [f(tₙ₊₂, w*ₙ₊₂) + 4f(tₙ₊₁, wₙ₊₁) + f(tₙ, wₙ)]

Explication :
- On utilise la prédiction w*ₙ₊₂ dans f(tₙ₊₂, w*ₙ₊₂)
- Ceci rend le schéma explicite (calculable)
- C'est une formule d'intégration à 3 points"

ÉTAPE 4 : Algorithme complet
"Algorithme Prédicteur-Correcteur :

INITIALISATION :
- w₀ = α (condition initiale)
- w₁ = β (calculé par une autre méthode, ex: Euler ou Taylor)

ITÉRATION (pour i = 0, 1, 2, ..., N-2) :
1. PRÉDICTEUR :
   w*ᵢ₊₂ = wᵢ + 2h × f(tᵢ₊₁, wᵢ₊₁)

2. CORRECTEUR :
   wᵢ₊₂ = wᵢ + (h/3) × [f(tᵢ₊₂, w*ᵢ₊₂) + 4f(tᵢ₊₁, wᵢ₊₁) + f(tᵢ, wᵢ)]

3. Incrémenter : i ← i + 1

FIN ITÉRATION"

ÉTAPE 5 : Analyse du choix
"Question : Le choix d'Euler comme Prédicteur est-il bon ?

Analyse :
- Correcteur d'ordre 4 (donné dans l'énoncé)
- Prédicteur Euler d'ordre 1

L'ordre global de la méthode Prédicteur-Correcteur est limité par le PLUS FAIBLE des deux, donc ordre 1.

Conclusion :
NON, le choix d'Euler (ordre 1) n'est pas optimal pour un Correcteur d'ordre 4.

Meilleur choix :
Il faudrait utiliser un Prédicteur d'ordre au moins 4, par exemple :
- Runge-Kutta d'ordre 4 (RK4)
- Adams-Bashforth d'ordre 4
- Taylor d'ordre 4

Ainsi, l'ordre global serait 4."

B) MÉTHODE DE TAYLOR D'ORDRE 2

Calcul préalable des dérivées partielles :
fₜ = ∂f/∂t = ... (montrer le calcul)
