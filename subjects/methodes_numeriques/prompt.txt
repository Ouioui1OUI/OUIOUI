MÉTHODOLOGIE DÉTAILLÉE

1. PROBLÈME DE CAUCHY : y'(t) = f(t,y), y(t₀) = y₀

Q1 : EXISTENCE ET UNICITÉ (Théorème Cauchy-Lipschitz)

Étapes OBLIGATOIRES :

a) Définir le domaine D
   - D = {(t,y) : a ≤ t ≤ b, y ∈ ℝ}
   - Vérifier que D est convexe (tube infini → toujours convexe)

b) Continuité de f
   - Décomposer f(t,y) en fonctions élémentaires
   - Conclure : "f est C∞ sur ℝ×ℝ, donc continue sur D"

c) Condition de Lipschitz (via Théorème 1)
   - Calculer ∂f/∂y = ...
   - Montrer |∂f/∂y| ≤ L sur D
   - Exemple : |∂f/∂y| = |t² cos(t)| ≤ t² ≤ 4 pour t ∈ [0,2]
   - Conclure : "f lipschitzienne par rapport à y de constante L = ..."

CONCLUSION : "Le problème admet une solution unique y(t) de classe C¹ sur [a,b]"

---

Q2 : SOLUTION EXACTE

Identifier le type d'EDO :
- Linéaire : y' + p(t)y = q(t)
- Séparable : y' = g(t)h(y)
- Bernoulli : y' + p(t)y = q(t)yⁿ
- Exacte

Pour EDO LINÉAIRE (le plus fréquent) :

a) Équation homogène
y' + p(t)y = 0
→ Solution : yₕ(t) = λ exp(-∫p(t)dt)

b) Facteur intégrant
μ(t) = exp(∫p(t)dt)
Calcul détaillé : ∫p(t)dt = ...
μ(t) = exp(...) = ...

c) Solution particulière (Variation de la Constante)
yₚ(t) = [∫μ(t)q(t)dt] / μ(t)
Calcul détaillé de l'intégrale : ...

d) Solution générale
y(t) = yₕ(t) + yₚ(t) = λ exp(...) + ...

e) Application de la condition initiale
y(t₀) = y₀
→ Résoudre pour λ : λ = ...

f) VÉRIFICATION OBLIGATOIRE
y'(t) = ... (dériver)
f(t,y(t)) = ... (substituer)
Vérifier : y'(t) = f(t,y(t)) ✓
Vérifier : y(t₀) = y₀ ✓

Q3 : PROBLÈME BIEN POSÉ (Stabilité)

Problème perturbé :
z'(t) = f(t,z) + δ(t), z(t₀) = y₀ + ε₀
|δ(t)| < ε, |ε₀| < ε

a) Résoudre le problème perturbé (même méthode que Q2)

b) Majorer |z(t) - y(t)|
|z(t) - y(t)| = |...| 
≤ |δ| × ... + |ε₀| × ... (inégalités triangulaires)
≤ K × ε avec K = ... (constante indépendante de ε et t)

CONCLUSION : "Le problème est bien posé : |z-y| ≤ K × ε"

2. MÉTHODES NUMÉRIQUES

RÈGLE CRITIQUE : TABLEAUX OBLIGATOIRES

Pour TOUTE méthode itérative, tu DOIS créer ce tableau :

| n | tₙ | sin(tₙ) | f(tₙ,wₙ) = ... | wₙ₊₁ = wₙ + ... | y(tₙ) exact | Erreur |y-w| |
|---|-----|----------|----------------|-----------------|-------------|---------------|
| 0 | 0.0 | 0.000    | 2×0 + 0 = 0.000| 2.0 (CI)        | 2.000       | 0.0000        |
| 1 | 0.1 | 0.0998   | 2×0.0998+... = ...| 2.0 + 0.1×0 = 2.0| 2.0300    | 0.0300        |

APRÈS le tableau, détaille 2-3 itérations ligne par ligne :

Itération n=0 :
- Donnée : t₀ = 0, w₀ = 2 (condition initiale)
- Calcul : sin(t₀) = sin(0) = 0
- f(t₀,w₀) = 2 × sin(0) + sin(0) = 0
- w₁ = w₀ + h × f(t₀,w₀) = 2 + 0.1 × 0 = 2.0

Itération n=1 :
- t₁ = t₀ + h = 0 + 0.1 = 0.1
- sin(t₁) = sin(0.1) = 0.09983 (calculatrice)
- f(t₁,w₁) = 2 × 0.09983 + 0.09983 = 0.2995
- w₂ = w₁ + h × f(t₁,w₁) = 2.0 + 0.1 × 0.2995 = 2.02995

---

A) MÉTHODE D'EULER

Formule :
tₙ = t₀ + n × h
w₀ = y₀
wₙ₊₁ = wₙ + h × f(tₙ, wₙ)

Ordre : Erreur globale O(h)

---

B) MÉTHODE DE TAYLOR D'ORDRE 2

Calcul des dérivées partielles OBLIGATOIRE :

fₜ = ∂f/∂t = ... (montrer le calcul)
fᵧ = ∂f/∂y = ... (montrer le calcul)

Formule :
wₙ₊₁ = wₙ + h × f(tₙ,wₙ) + (h²/2) × [fₜ + fᵧ × f]

Ordre : Erreur globale O(h²)

---

C) RUNGE-KUTTA D'ORDRE 2 (Heun)

Formule :
wₙ₊₁ = wₙ + (h/2) × [f(tₙ,wₙ) + f(tₙ₊₁, wₙ + h × f(tₙ,wₙ))]

Ordre : Erreur globale O(h²)

---

D) RUNGE-KUTTA D'ORDRE 4 (RK4)

Formule :
k₁ = h × f(tₙ, wₙ)
k₂ = h × f(tₙ + h/2, wₙ + k₁/2)
k₃ = h × f(tₙ + h/2, wₙ + k₂/2)
k₄ = h × f(tₙ + h, wₙ + k₃)
wₙ₊₁ = wₙ + (k₁ + 2k₂ + 2k₃ + k₄)/6

Créer une colonne pour CHAQUE kᵢ dans le tableau :

| n | tₙ | k₁ | k₂ | k₃ | k₄ | wₙ₊₁ | y(tₙ) | Erreur |
|---|----|----|----|----|----|----- |-------|--------|

Ordre : Erreur globale O(h⁴)
