2. INTERPOLATION POLYNOMIALE - NEWTON

STRUCTURE OBLIGATOIRE (basée sur le corrigé officiel)

Question : "Retrouver le polynôme d'interpolation par la Méthode de Newton."

ÉTAPE 1 : Objectif

ÉTAPE 2 : Formule de Newton
"Formule de Newton :
P(x) = a₀ + a₁(x - x₀) + a₂(x - x₀)(x - x₁) + a₃(x - x₀)(x - x₁)(x - x₂)

8. POLYNÔME D'INTERPOLATION DE LAGRANGE SUR 3 POINTS

Question : "Écrire le polynôme d'interpolation de Lagrange P(s) associé aux 3 points {(ti, fi), (ti+1, fi+1), (ti+2, fi+2)}."

STRUCTURE OBLIGATOIRE :

"Les 3 points sont : (ti, fi), (ti+1, fi+1), (ti+2, fi+2)
où fi = f(ti, y(ti)), etc.

Avec h = ti+1 - ti (pas constant), on a :
- ti+1 = ti + h
- ti+2 = ti + 2h

Formule de Lagrange :
P(s) = fi × Li(s) + fi+1 × Li+1(s) + fi+2 × Li+2(s)

où :
Li(s) = [(s - ti+1)(s - ti+2)] / [(ti - ti+1)(ti - ti+2)]
      = [(s - ti+1)(s - ti+2)] / [(-h)(-2h)]
      = [(s - ti+1)(s - ti+2)] / (2h²)

Li+1(s) = [(s - ti)(s - ti+2)] / [(ti+1 - ti)(ti+1 - ti+2)]
        = [(s - ti)(s - ti+2)] / [(h)(-h)]
        = -[(s - ti)(s - ti+2)] / h²

Li+2(s) = [(s - ti)(s - ti+1)] / [(ti+2 - ti)(ti+2 - ti+1)]
        = [(s - ti)(s - ti+1)] / [(2h)(h)]
        = [(s - ti)(s - ti+1)] / (2h²)

Polynôme complet :
P(s) = fi × [(s-ti+1)(s-ti+2)]/(2h²) - fi+1 × [(s-ti)(s-ti+2)]/h² + fi+2 × [(s-ti)(s-ti+1)]/(2h²)"

9. PRÉDICTEUR-CORRECTEUR

Question : "Écrire le schéma Prédicteur (Euler) pour prédire wi+2."

**STRUCTURE BASÉE SUR LE CORRIGÉ OFFICIEL** :

ÉTAPE 1 : Objectif
"Objectif : Adapter la méthode d'Euler pour prédire la valeur wᵢ₊₂ nécessaire au schéma implicite."

ÉTAPE 2 : Analyse
"Analyse :
Le schéma correcteur relie wᵢ₊₂ à wᵢ et aux dérivées en i, i+1, i+2. Il est implicite car wᵢ₊₂ apparaît dans f(tᵢ₊₂, wᵢ₊₂).

Pour estimer wᵢ₊₂, nous utilisons une méthode explicite.
Nous sommes à l'étape où wᵢ et wᵢ₊₁ sont connus.

La méthode d'Euler standard partant de tᵢ₊₁ permet d'atteindre tᵢ₊₂."

ÉTAPE 3 : Formule du Prédicteur
"Formule du Prédicteur (Euler à partir de i+1) :
wᵢ₊₂⁽ᴾ⁾ = wᵢ₊₁ + h × f(tᵢ₊₁, wᵢ₊₁)

Note : On utilise le point le plus proche (i+1) pour la précision."

---

Question : "Écrire l'algorithme Prédicteur-Correcteur."

STRUCTURE BASÉE SUR LE CORRIGÉ OFFICIEL :

"Algorithme :
Pour chaque étape i allant de 0 à N-2 :

1. Prédiction (P) :
   Calculer une valeur provisoire de wᵢ₊₂
   wᵢ₊₂* = wᵢ₊₁ + h × f(tᵢ₊₁, wᵢ₊₁)

2. Évaluation (E) :
   Évaluer la fonction f avec cette prédiction
   fᵢ₊₂* = f(tᵢ₊₂, wᵢ₊₂*)

3. Correction (C) :
   Appliquer la formule de Simpson
   wᵢ₊₂ = wᵢ + (h/3) × [fᵢ₊₂* + 4f(tᵢ₊₁, wᵢ₊₁) + f(tᵢ, wᵢ)]

Initialisation :
Le schéma nécessite w₀ (donné, α) et w₁ (donné, β)."

---

Question : "Le choix d'Euler comme Prédicteur est-il bon ?"

**STRUCTURE BASÉE SUR LE CORRIGÉ OFFICIEL** :

"Question : Le schéma correcteur (Simpson) est d'ordre 4. L'utilisation d'Euler (ordre 1) comme prédicteur est-elle un bon choix ?

Analyse :
Le théorème sur l'ordre des méthodes Prédicteur-Correcteur stipule que si le correcteur est d'ordre q et le prédicteur d'ordre p, l'ordre global de la méthode (en mode PEC ou PECE sans itération jusqu'à convergence) est min(q, p+1).

Application aux données :
- Correcteur (Simpson) : Ordre q = 4
- Prédicteur (Euler) : Ordre p = 1

Ordre résultant = min(4, p+1) = min(4, 1+1) = 2.

Conclusion :
Non, la méthode d'Euler n'est pas un bon choix car elle dégrade l'ordre global de la méthode à 2, nous faisant perdre la précision d'ordre 4 offerte par Simpson.

Pour conserver l'ordre 4, il faudrait choisir un prédicteur tel que p + 1 ≥ 4, c'est-à-dire p ≥ 3.

Un meilleur choix serait un prédicteur d'ordre 3 ou 4, comme par exemple la méthode d'Adams-Bashforth à 3 ou 4 pas, ou le prédicteur de Milne (ordre 4)."
