1. TESTS D'HYPOTHÈSES

STRUCTURE OBLIGATOIRE POUR TEST SUR UNE MOYENNE

Exemple : "Peut-on conclure avec un risque de ... que ... ?"

ÉTAPE 1 : Reformuler la question
"On cherche à tester si le volume moyen des bouteilles est différent de 1 litre au seuil α = 4%."

ÉTAPE 2 : Hypothèses
"Hypothèses :
H₀ : μ = μ₀ (le volume moyen est de 1 litre)
H₁ : μ ≠ μ₀ (le volume moyen est différent de 1 litre)

Test bilatéral au seuil α = ...%"

ÉTAPE 3 : Données
"Données :
- Taille échantillon : n = ...
- Moyenne échantillon : x̄ = ...
- Écart-type population : σ = ... (connu)
- Moyenne théorique : μ₀ = ..."

ÉTAPE 4 : Statistique de test
"Statistique de test :
Comme σ est connu et n ≥ 30 [ou n < 30 mais distribution normale], on utilise la loi normale :
Z = (x̄ - μ₀) / (σ/√n)

Calcul :
Z = (... - ...) / (.../√...)
Z = ... / (.../...)
Z = ... / ...
Z = ..."

ÉTAPE 5 : Valeur critique
"Valeur critique :
Pour un test bilatéral au seuil α = ...%, on cherche z_α/2 tel que P(|Z| > z_α/2) = α

α = ...% = ...
α/2 = .../2 = ...

Table normale : z_α/2 = z_... = ...
[OU] Avec Excel : =LOI.NORMALE.STANDARD.INVERSE(1-α/2) = ..."

ÉTAPE 6 : Règle de décision
"Règle de décision :
- Si |Z_obs| > z_α/2 : on rejette H₀
- Si |Z_obs| ≤ z_α/2 : on ne rejette pas H₀

Comparaison :
|Z_obs| = |...| = ...
z_α/2 = ...

|...| [> ou ≤] ... : on [rejette / ne rejette pas] H₀"

ÉTAPE 7 : Conclusion
"Conclusion :
Au seuil de risque α = ...%, on [peut / ne peut pas] conclure que le volume moyen des bouteilles est différent de ... litre.
[Explication en contexte]"

ÉTAPE 8 : P-value (si demandée)
"Calcul de la p-value :
p-value = 2 × P(Z > |Z_obs|) (test bilatéral)
p-value = 2 × P(Z > ...)
p-value = 2 × (1 - Φ(...))
p-value = 2 × (1 - ...)
p-value = 2 × ...
p-value = ... = ...%

Interprétation :
- Si p-value < α : on rejette H₀

... [< ou ≥] ... : on [rejette / ne rejette pas] H₀

À partir du seuil α = ...%, on peut conclure que la prétention est fausse."

2. RÉGRESSION LINÉAIRE SIMPLE

STRUCTURE OBLIGATOIRE

Question : "Calculer les coefficients de la droite de régression linéaire"

ÉTAPE 1 : Modèle
"Modèle de régression linéaire simple :
yᵢ = β₀ + β₁xᵢ + εᵢ"

ÉTAPE 2 : Hypothèses (si demandées)
"Hypothèses sur le terme d'erreur εᵢ :
1. E(εᵢ) = 0 : les erreurs sont d'espérance nulle
2. Var(εᵢ) = σ² : les erreurs ont une variance constante (homoscédasticité)
3. Cov(εᵢ, εⱼ) = 0 pour i ≠ j : les erreurs sont non corrélées
4. [Si normalité requise] εᵢ ~ N(0, σ²) : les erreurs suivent une loi normale"

ÉTAPE 3 : Données fournies
"Données fournies :
- Nombre d'observations : n = ...
- Moyennes : x̄ = ..., ȳ = ...
- Somme des carrés : Σ(xᵢ - x̄)² = ...
- Somme des carrés : Σ(yᵢ - ȳ)² = ...
- Somme des produits croisés : Σ(xᵢ - x̄)(yᵢ - ȳ) = ..."

ÉTAPE 4 : Calcul de β₁
"Calcul de β₁ (estimateur des moindres carrés) :

Formule :
β̂₁ = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²

Calcul :
β̂₁ = ... / ...
β̂₁ = ...

Interprétation : Quand [nom variable X] augmente de 1 unité, [nom variable Y] [augmente/diminue] en moyenne de |β̂₁| = ... unités."

ÉTAPE 5 : Calcul de β₀
"Calcul de β₀ (estimateur des moindres carrés) :

Formule :
β̂₀ = ȳ - β̂₁ × x̄

Calcul :
β̂₀ = ... - (...) × (...)
β̂₀ = ... - ...
β̂₀ = ...

Interprétation : Quand [nom variable X] = 0, [nom variable Y] vaut en moyenne β̂₀ = ... [si pertinent dans le contexte]."

ÉTAPE 6 : Modèle estimé
"Expression du modèle estimé :
ŷ = β̂₀ + β̂₁ × x
ŷ = ... + (...) × x
ŷ = ... + ... x [ou] ŷ = ... - ... x (si β̂₁ < 0)"

---

ÉTAPE 7 : SCE (si demandée)
"Expression de la SCE en fonction de β̂₁ et Σ(xᵢ - x̄)² :

SCE = Σ(ŷᵢ - ȳ)²
Or, ŷᵢ = β̂₀ + β̂₁xᵢ et β̂₀ = ȳ - β̂₁x̄
Donc : ŷᵢ - ȳ = (β̂₀ + β̂₁xᵢ) - ȳ
                = (ȳ - β̂₁x̄ + β̂₁xᵢ) - ȳ
                = β̂₁(xᵢ - x̄)

Par conséquent :
SCE = Σ[β̂₁(xᵢ - x̄)]²
SCE = β̂₁² × Σ(xᵢ - x̄)²

Calcul :
SCE = (...)² × ...
SCE = ... × ...
SCE = ..."

ÉTAPE 8 : Coefficient R² (si demandé)
"Calcul du coefficient de détermination R² :

Formule :
R² = SCE / SCT

Avec :
SCT = Σ(yᵢ - ȳ)² = ... (donné)
SCE = β̂₁² × Σ(xᵢ - x̄)² = ... (calculé ci-dessus)

Calcul :
R² = ... / ...
R² = ... = ...% (en pourcentage)

Interprétation : 
Le modèle explique ...% de la variance de [nom variable Y].
[Si R² > 0.7] La qualité de l'ajustement est [bonne/moyenne/faible].
[Explication contextualisée]"

Question : "Que signifie 'Residual standard error'. En déduire SCR, SCT, SCE."

ÉTAPE 1 : Définition
"Le 'Residual standard error' (erreur résiduelle standard) est l'écart-type résiduel s :

s = √[SCR / (n - p)]

D'après la sortie R :
s = ... (valeur fournie)
n = ... (nombre d'observations)
p = ... (nombre de coefficients β)
ddl = n - p = ..."

ÉTAPE 2 : SCR
"Calcul de la SCR (Somme des Carrés Résiduels) :
s² = SCR / (n - p)
SCR = s² × (n - p)
SCR = (...)² × (...)
SCR = ... × ...
SCR = ..."

ÉTAPE 3 : SCT et SCE
"D'après le coefficient R² :
R² = SCE / SCT = ...
Donc : SCE = R² × SCT

Aussi : SCT = SCE + SCR
Donc : SCT = SCE + ...

Et : SCE = R² × SCT
Donc : SCT = SCE + SCR
       SCT = R² × SCT + SCR
       SCT × (1 - R²) = SCR
       SCT = SCR / (1 - R²)

Calcul :
SCT = ... / (1 - ...)
SCT = ... / ...
SCT = ...

SCE = SCT - SCR
SCE = ... - ...
SCE = ...

Vérification :
R² = SCE / SCT = ... / ... = ... ✓"
