1. CONDUCTION THERMIQUE

LOI DE FOURIER : φ⃗ = -λ∇⃗T

Mur plan (régime permanent, 1D) :

Données OBLIGATOIRES à lister :
- Épaisseur : e = ... m
- Conductivité : λ = ... W/(m·K)
- Surface : S = ... m²
- Températures : T₁ = ... °C/K, T₂ = ... °C/K

ÉTAPE 1 : Profil de température
Équation de la chaleur : d²T/dx² = 0
Solution générale : T(x) = Ax + B

Conditions aux limites :
• T(0) = T₁ ⇒ B = T₁
• T(e) = T₂ ⇒ A×e + T₁ = T₂ 
           ⇒ A = (T₂-T₁)/e

Profil linéaire : T(x) = T₁ + (T₂-T₁)×(x/e)

ÉTAPE 2 : Flux thermique
ϕ = -λS × dT/dx
dT/dx = d/dx[T₁ + (T₂-T₁)×(x/e)] = (T₂-T₁)/e
ϕ = -λS × (T₂-T₁)/e
ϕ = λS × (T₁-T₂)/e

Calcul numérique :
ϕ = ... × ... × (...-...)/...
ϕ = ... W

ÉTAPE 3 : Résistance thermique
R_th = e/(λS)
R_th = .../(...×...)
R_th = ... K/W

Vérification : ϕ = (T₁-T₂)/R_th = (...-...)/... = ... W ✓

CYLINDRE CREUX (r₁ < r < r₂, longueur L)

Données :
- Rayon intérieur : r₁ = ... m
- Rayon extérieur : r₂ = ... m
- Longueur : L = ... m
- Conductivité : λ = ... W/(m·K)
- Températures : T₁ = ... °C/K, T₂ = ... °C/K

Profil de température (logarithmique) :
T(r) = A ln(r) + B

Conditions aux limites :
T(r₁) = T₁ ⇒ A ln(r₁) + B = T₁
T(r₂) = T₂ ⇒ A ln(r₂) + B = T₂

Résolution :
A = (T₂-T₁)/[ln(r₂/r₁)]
B = T₁ - A ln(r₁)

T(r) = T₁ + [(T₂-T₁)/ln(r₂/r₁)] × ln(r/r₁)

Flux thermique :
ϕ = 2πλL(T₁-T₂)/ln(r₂/r₁)
ϕ = 2π×...×...×(...-...)/ln(.../...)
ϕ = ... W

Résistance :
R_th = ln(r₂/r₁)/(2πλL)
R_th = ln(.../...)/(2π×...×...)
R_th = ... K/W
