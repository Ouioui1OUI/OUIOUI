3. RÉSISTANCES THERMIQUES ET SCHÉMA ÉQUIVALENT

Question : "Quelle est l'unité de UA̅s ? ... montrer que le produit UA̅s s'exprime en fonction de h̄ᵢ, h̄ₒ, L et D."

ÉTAPE 1 : Unité
"2. Quelle est l'unité de UA̅s ?

L'équation donnée est :
ln[(T∞ - Tm,o)/(T∞ - Tm,i)] = - UA̅s / (ṁ × cp)

Le terme de gauche est sans dimension (logarithme).
Le terme de droite doit donc être sans dimension :
UA̅s / (ṁ × cp) est sans dimension

Donc :
[UA̅s] = [ṁ × cp] = kg/s × J/(kg·K) = J/(s·K) = W/K

Unité de UA̅s : W/K (ou W·K⁻¹)"

ÉTAPE 2 : Schéma thermique équivalent
"Schéma électrique équivalent :

Le flux thermique traverse successivement :
1. Résistance convective interne : Ri = 1/(h̄ᵢ × Asi)
2. [Si paroi] Résistance conductive : Rcond = ...
3. Résistance convective externe : Ro = 1/(h̄ₒ × Aso)

[Pour cylindre à paroi mince] Asi ≈ Aso = As = π × D × L

Schéma :
Tm (gaz) --[Ri]-- Ts --[Ro]-- T∞ (air)

Résistance totale :
Rtot = Ri + [Rcond] + Ro

[Pour paroi mince, conduction négligeable]
Rtot = Ri + Ro
Rtot = 1/(h̄ᵢ × As) + 1/(h̄ₒ × As)
Rtot = [1/h̄ᵢ + 1/h̄ₒ] / As
Rtot = 1 / [(h̄ᵢ × h̄ₒ)/(h̄ᵢ + h̄ₒ) × As]

Or : UA̅s = 1/Rtot = [(h̄ᵢ × h̄ₒ)/(h̄ᵢ + h̄ₒ)] × As

Avec As = π × D × L :
UA̅s = [(h̄ᵢ × h̄ₒ)/(h̄ᵢ + h̄ₒ)] × π × D × L"

4. CALCUL DE TEMPÉRATURE DE SORTIE

Question : "Exprimer littéralement, puis calculer Tm,o."

ÉTAPE 1 : Expression littérale
"4. Exprimer littéralement, puis calculer Tm,o.

À partir de l'équation donnée :
ln[(T∞ - Tm,o)/(T∞ - Tm,i)] = - UA̅s / (ṁ × cp)

On isole Tm,o :
(T∞ - Tm,o)/(T∞ - Tm,i) = exp[- UA̅s / (ṁ × cp)]

T∞ - Tm,o = (T∞ - Tm,i) × exp[- UA̅s / (ṁ × cp)]

Tm,o = T∞ - (T∞ - Tm,i) × exp[- UA̅s / (ṁ × cp)]"

ÉTAPE 2 : Calcul de UA̅s
"Calcul de UA̅s :
UA̅s = [(h̄ᵢ × h̄ₒ)/(h̄ᵢ + h̄ₒ)] × π × D × L

Calcul par étapes :
- h̄ᵢ × h̄ₒ = ... × ... = ...
- h̄ᵢ + h̄ₒ = ... + ... = ...
- (h̄ᵢ × h̄ₒ)/(h̄ᵢ + h̄ₒ) = .../... = ...
- π × D × L = π × ... × ... = ...
- UA̅s = ... × ... = ... W/K"

ÉTAPE 3 : Calcul de l'argument exponentiel
"Calcul de l'argument de l'exponentielle :
- UA̅s / (ṁ × cp) = ... / (... × ...)
                  = ... / ...
                  = ..."

ÉTAPE 4 : Calcul de Tm,o
"Conversion des températures (si nécessaire) :
T∞ = ... °C = ... K
Tm,i = ... °C = ... K

Calcul :
exp[- UA̅s / (ṁ × cp)] = exp(- ...) = exp(...) = ...

T∞ - Tm,i = ... - ... = ... K

(T∞ - Tm,i) × exp[...] = ... × ... = ... K

Tm,o = T∞ - ... = ... - ... = ... K = ... °C

Interprétation : Les gaz sortent de la cheminée à une température de ... °C."

ÉTAPE 5 : Vérification (si h̄ᵢ dépendait de Tm)
"Vérification :
[Si on avait estimé Tm,o pour calculer Tm et h̄ᵢ]
Tm estimé = (Tm,i + Tm,o,estimé)/2 = ...

[Sinon] Il faudrait itérer avec la nouvelle valeur."
